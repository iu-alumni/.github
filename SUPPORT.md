## Getting Help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/iu-alumni) | Bug reports, feature requests (use templates) |
| [Telegram community](https://t.me/+8hrAOuObPXQzZGRi) | Quick questions, general discussion |
| Issue label `question` | Open-ended questions that belong in the repo context |

---

## Before Asking

1. Check the [documentation](https://iu-alumni.github.io/docs) for the relevant component.
2. Search open and closed issues in the relevant repository.
3. Make sure you are on the latest version of `main`.

---

## Security Issues

**Do not open a public issue for security vulnerabilities.**  
Read the [Security Policy](SECURITY.md) for responsible disclosure instructions.
