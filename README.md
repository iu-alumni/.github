# iu-alumni/.github

This repository contains org-wide GitHub community health files and the organization profile page.

## Contents

| File | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | Organization homepage shown at [github.com/iu-alumni](https://github.com/iu-alumni) |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Default contributing guide inherited by all repos without their own |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Contributor Covenant — applies across the org |
| [`SECURITY.md`](SECURITY.md) | Vulnerability reporting policy and response timeline |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |

## How GitHub uses these files

Files placed here serve as **org-wide defaults**. Any repository in the `iu-alumni` organization that does not have its own copy of `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, or `SUPPORT.md` will automatically inherit the version from this repo.

The `profile/README.md` file is rendered as the public-facing organization page.
